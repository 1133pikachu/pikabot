# pikabot-dev
ぴかさばおよびぴかの身内鯖で使いやすいように作成しているbotです。

## 前提要件
・discord.js v14.x.x以上<br>
・node.js v20.x.x以上

## 使用方法
### リポジトリのクローン
```
git clone https://github.com/1133pikachu/pikabot.git
```
### configの作成
config-example.jsonをconfig.jsonにrenameします<br>
各種token,prefixを記入します
### モジュールのインストール
```
npm i
```
### サーバーにスラッシュコマンドを登録
```
npm deploy
```
### botの起動
```
npm start
```
